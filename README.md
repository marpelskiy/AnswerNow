# AnswerNow
Легкое расширение для Chrome, которое делает скриншот активной вкладки и мгновенно решает задачи с помощью ИИ. Никакого мусора, только захват экрана и ответ. Переписано с нуля для стабильной работы и обхода лимитов.
1. Перейдите в chrome://extensions/
2. Включите Developer Mode
3. Нажмите 'Load unpacked' и выберите папку с проектом
